# Increasing Cost Efficiency for Digital Advertising

## Project Overview
This project demonstrates a full data science workflow:
- Cleaning & Merging Datasets
- Exploratory Data Analysis (EDA)
- Predictive modeling (Logistic Regression & Light GBM)
- Model comparison and evaluation
- Tableau dashboard and stakeholder presentation

## Tech Stack
- Python (pandas, scikit-learn, matplotlib)
- Tableau (visual dashboards and stakeholder presentation)
- Jupyter Notebook (analysis + documentation)
- Streamlit (interactive demostration)
